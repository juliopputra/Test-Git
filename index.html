BELAJAR GIT BINAR !

command untuk configurasi

  • git config --global user.name "your name" -> set user name
  • git config --global user.email "your@mail.com" -> set user email
  • git config --glboal core.editor "vim" -> set default editor untuk git
  • git config --list -> untuk melihat git config

command ketika masih di local computer kita

  • git init --> Initialize Directory Menggunakan Git ( Work Directory )
  • git status --> Checking Status File ( Active Status )
  • git add file_name --> Add File ke Staging ( Staging )
  • git commit -m 'message' --> Saving to Local Repo ( Local Repository )
  • git commit -a -> untuk menulis commit message di default editor ( tambah commit pesan di dalam editor )
  • git commit -am 'commit message' -> short command untuk langsung commit file ( commit file keseluruhan, tanpa perlu add dulu )
  • git log --> Untuk Mengecek History Perubahan ( Cek Perubahan )
  • git log --oneline -> melihat log dalam versi singkat ( simple log dengan commit massage )
  • git restore --staged file_name --> Unstaged File ( Mengeluarkan file dari Staging Flow )
  • git restore file_name --> Restore ke commit point sebelumnya ( Restore Point )
  • git reset --hard HEAD~ --> Rollback Commit Keluar Staging ( Mengeluarkan file dari Staging Flow )
  • git rest --soft HEAD~ --> Rollback ke Commit Sebelumnya di Staging ( Mengembalikan file ke posisi Staging Flow Sebelumnya )

command kalau sudah bekerja dengan remote repository

  • git remote add remote_name git_address --> ( Create Remote for Github / Gitlab Repo )
  • git push remote_name local_repo_branch --> ( Push to Repo Github / Gitlab )
  • git pull remote_name branch_name -> ( untuk mengambil perubahan dari remote )
  • git fetch remote_name -> ( untuk mensyncronkan remote repository dan local repository )
